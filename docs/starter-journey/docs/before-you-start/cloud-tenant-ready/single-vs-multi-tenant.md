# Single vs Multi tenant

Choose between a single tenant for your organization or multiple tenants (e.g. by team or division).

For tenant setup you have:

- **[Single-tenant setup](/docs/before-you-start/cloud-tenant-ready/single-tenant-setup)** – One tenant for your organization
- **[Multi-tenant setup](/docs/before-you-start/cloud-tenant-ready/multi-tenant-setup)** – Multiple tenants
