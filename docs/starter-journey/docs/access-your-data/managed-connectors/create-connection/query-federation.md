# Query federation

:::info
* Use Lakehouse federation with your connection.
* ✅ A Foreign Catalog is required if pushdown queries are required on the data pipeline.
* ❌ A Foreign Catalog is not required if you plan to create an ingestion pipeline. The connection is enough.
:::
