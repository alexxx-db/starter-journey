# MCP Skills

> **You'll learn** how to use MCP skills to build pipelines with AI-assisted tooling.
>
> **Prereqs:** [Build the first pipeline](/docs/build-first-pipeline)

## Why this matters

MCP (Model Context Protocol) skills let AI agents interact directly with Databricks APIs — creating pipelines, deploying resources, and running queries through natural-language instructions. This approach removes the gap between describing what you want and having it deployed.
