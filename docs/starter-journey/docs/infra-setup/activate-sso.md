Enable and configure Single Sign-On (SSO) for Databricks. Configure SSO so users can sign in with your identity provider (IdP) using single sign-on.
:::

:::warning Azure
Not required, Entra ID is enabled by default.
:::

:::success AWS
- [Configure SSO on your identity provider](https://docs.databricks.com/aws/en/security/auth/single-sign-on/).
:::

:::note GCP
- Check if Google SSO is enabled by default, if not follow the guide:
- [Configure SSO on your identity provider](https://docs.databricks.com/aws/en/security/auth/single-sign-on/).
:::
