# AWS – Manual

:::info
Create a Databricks workspace on AWS manually in a couple clicks. Follow the PDF guide.
:::

:::info Best practices and recommendation
* Start with the **development** workspace.
  * Repeat for **staging** and **production** workspaces.
:::

:::success Guide
* Select the targeted AWS account when prompted.
* Make sure to deploy the workspace on the region where your data sources are to avoid egress fees.
* <a target="_blank" href={useBaseUrl('/pdfs/AWS-Automated-Configuration-Classic-Workspace-Deployment.pdf')}>**PDF - AWS Automated Configuration Classic Workspace Deployment**</a>
:::
