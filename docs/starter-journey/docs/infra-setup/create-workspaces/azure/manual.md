# Azure – Manual

:::info
Create a Databricks workspace on Azure manually.
:::

:::info Best practices and recommendation
- Follow the video to deploy the first workspace (**development** workspace).
- Take in consideration the subscription and resource group where it's supposed to be deployed.
  - i.e dev workspace on the dev subscription and dev databricks resource group.
- Repeat for **staging** and **production** workspaces. 
:::
