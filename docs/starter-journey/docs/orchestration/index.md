## Get Started

In your data+AI project, every aspect related to:

* Orchestration.
* Tasks, DAGs and dependencies.
* Schedule of execution and triggers.
* Retries and notifications (success, failures).

Should be setup as one or multiple [Lakeflow Jobs (AKA Databricks Jobs)](https://docs.databricks.com/aws/en/jobs).

## Create the first job

- **[UI](/docs/orchestration/ui)** – Orchestrate workflows using the Databricks UI
- **[DABs](/docs/orchestration/dabs)** – Orchestrate workflows using Databricks Asset Bundles
- **[MCP skills](/docs/orchestration/mcp-skills)** – Orchestrate workflows using MCP skills
